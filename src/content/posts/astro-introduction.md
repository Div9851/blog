# Astro を使ってみた感想

このブログは Astro というフレームワークで構築しています。

## Astro とは

Astro は、コンテンツにフォーカスした高速な Web サイトを構築するためのフレームワークです。

### 特徴

- **高速**: デフォルトで JavaScript をゼロに
- **柔軟**: React、Vue、Svelte など好きな UI フレームワークを使用可能
- **シンプル**: 学習曲線が緩やか

## 使ってみての感想

非常にシンプルで使いやすいフレームワークだと感じました。
特にコンテンツコレクション機能が便利です。
