# Cloudflare Pages でデプロイ

このブログは Cloudflare Pages でホスティングしています。

## Cloudflare Pages とは

Cloudflare Pages は、静的サイトや JAMstack サイトをホスティングできるプラットフォームです。

## メリット

- **無料**: 個人プロジェクトなら無料で使える
- **高速**: Cloudflare のグローバルネットワークを利用
- **簡単**: GitHub と連携して自動デプロイ

## デプロイの流れ

1. GitHub にコードをプッシュ
2. Cloudflare Pages でプロジェクトを作成
3. ビルド設定を行う
4. 自動的にデプロイされる

非常に簡単にデプロイできるのでおすすめです。
