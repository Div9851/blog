# Markdown で書くコツ

ブログ記事を Markdown で書く際のコツをまとめました。

## 基本的な書き方

### 見出し

```markdown
# H1 見出し

## H2 見出し

### H3 見出し
```

### リスト

```markdown
- 箇条書き
- 項目 2
```

### リンク

```markdown
[リンクテキスト](URL)
```

## おすすめのエディタ

- VS Code
- Typora
- Obsidian

## まとめ

Markdown はシンプルで書きやすく、ブログ執筆に最適な形式です。
